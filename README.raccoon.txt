Name of the procesing script: Evaluate_Racoon_Life.py

Description: The purpose of this script is to read a comma separated data file that describes a racoon behavior, sort its contents and add extra information based on the given instructions. The script opens the file from a specific directory given by the user. Then it stores all the information into a dictionary with keys the headers of the opened file and values the data under each header. Additionally, this program calculate the average values of the position, the distance traveled by the racoon at each step and the total distance of the travel of the racoon. In the end, the script writes the output file in the Tab separated format requested by the user.

Input file: 2008Male00006.txt
Output file: Georges_life.txt
